주제 변경 정리

# 1. 기존 주제와 변경 주제

## 기존 주제: 어떤 고객에게 어떤 프로모션을 주면 실제 구매로 이어지는가?
- 프로모션 → 구매를 연결해야 하는 주제
- 단순히 오퍼를 받았는지, 봤는지가 아니라 그 오퍼 때문에 실제 transaction이 발생했는지까지 봐야 함

- 목표
    - 이 프로모션이 구매를 유도했다
    - 이 고객군에 이 오퍼를 주면 매출이 오른다
    - 이 오퍼가 실제로 돈이 된다

## 변경 주제: 어떤 고객 세그먼트가 어떤 프로모션에 더 잘 반응하는가?
- 프로모션 → 반응을 보는 주제
- received → viewed → completed 흐름 안에서 누가 더 잘 반응하는지를 분석함

- 목표 
    - 이 고객군은 특정 오퍼를 더 잘 본다
    - 이 고객군은 완료율이 높다
    - 이 오퍼 유형은 반응률이 높다
    - 이 고객군을 우선 타겟으로 삼는 전략이 유효해 보인다

---

# 2. 주제를 바꿔야 하는 이유
1. 거래가 어떤 프로모션 때문인지 알 수 없음\
transaction에는 구매 금액은 있으나 어떤 offer 때문에 구매했는지 연결하는 키가 없음\
이 오퍼를 받아서 산 건지, 원래 살 사람이 산 건지 구분하기 어렵다.

2. 통제군이 사실상 없음\
대부분 고객이 이미 프로모션을 받은 상태라서 프로모션을 안 받은 고객과 비교하기가 거의 불가능함\
프로모션이 실제 효과가 있었는지 원래도 구매할 고객이었는지 비교해서 말하기 어렵다.

3. ROI나 순이익 계산이 어려움\
ROI를 계산하려면 실제비용, 프로모션 원가 같은 비용정보가 필요하나 없다.\
그래서 **"이 프로모션이 실제로 돈이 된다"**까지 해석하기 어렵다.

4. 결론\
고객 반응 패턴 분석은 가능하지만\
프로모션이 실제 구매를 만들었는지 증명하는 분석에는 한계가 크다.

> 구매 효과 분석 → 반응 패턴 분석

# 3. 기존 주제와 비교
| 구분      | 기존 주제                              | 변경 주제                                             |
| ------- | ---------------------------------- | ------------------------------------------------- |
| 핵심 질문   | 어떤 프로모션이 실제 구매를 만들었는가              | 어떤 고객이 어떤 프로모션에 더 잘 반응하는가                         |
| 분석 흐름   | 프로모션 → 구매                          | 프로모션 → 반응                                         |
| 중심 이벤트  | transaction 중심                     | received / viewed / completed 중심                  |
| 해석 방식   | 효과 검증, 구매 유도, 매출 기여                | 패턴 설명, 반응 차이, 타겟 전략                               |
| 대시보드 목적 | 어떤 프로모션이 실제 구매를 만들었는지 판단           | 고객 세그먼트별 반응 차이를 보고 타겟팅 방향 도출                      |
| 메인 KPI  | 구매 전환율, 총 거래 금액, 고객당 평균 거래 금액, ROI | 오퍼 열람률, 오퍼 완료율, 세그먼트별 반응률, 오퍼 유형별 반응률, 채널 조합별 반응률 |

---

# 4. 변경된 주제에서 보는것
- 누가 오퍼를 받았는가?
- 누가 오퍼를 보았는가?
- 누가 오퍼를 완료했는가?
- 어떤 오퍼 유형이 반응이 좋은가?
- 어떤 고객 세그먼트가 더 반응이 좋은가?

> 원인 증명보다는 패턴 설명과 전략 제안에 초점을 둔다.

---

# 5. 변경된 주제의 대시보드 방향
고객 세그먼트별 프로모션 반응 차이를 시각화하고, 타겟팅 방향을 잡는 대시보드

- 고객 세그먼트별 열람률 비교
- 고객 세그먼트별 완료율 비교
- 오퍼 유형별 반응률 비교
- 채널 조합별 반응률 비교
- 반응이 높은 고객군과 오퍼 조합 파악

---

# 6. 변경된 주제의 메인 KPI 및 대시보드
1. 전체 반응 퍼널
    - 오퍼 수신 수
    - 오퍼 열람 수 / 열람률
    - 오퍼 완료 수 / 완료율

2. 고객 세그먼트별 반응 비교
    - 성별별 열람률 / 완료율
    - 연령대별 열람률 / 완료율
    - 소득대별 열람률 / 완료율
    - 가입 시점별 반응 차이
    - 신규/기존 고객 비교

3. 오퍼 유형 / 채널별 반응 비교
    - bogo, discount, informational별 열람률
    - 오퍼 유형별 완료율
    - reward / difficulty / duration에 따른 반응 차이
    - 채널 조합별 반응
        - email 포함 오퍼 vs 미포함 오퍼
        - mobile 포함 오퍼 vs 미포함 오퍼
        - social 포함 오퍼 vs 미포함 오퍼
        - 채널 조합별 열람률 / 완료율

4. 타겟팅 전략 인사이트
    - 어떤 고객군이 어떤 오퍼에 가장 잘 반응하는가?
    - 열람은 높은데 완료는 낮은 그룹은 어디인가?
    - 반응이 낮은 세그먼트는 어디서 이탈하는가?
    - 다음 캠페인에서 우선 공략할 고객군은 누구인가?

>구매 효과 증명보다 반응 구조와 타겟팅 전략 도출에 집중한다.
