# 내일배움캠프 `sparta-data11-practical-project`
데이터 분석 및 머신러닝 실전 프로젝트 저장소입니다.

## 프로젝트 개요
- 프로젝트명: 데이터 분석 및 머신러닝 실전 프로젝트
- 프로젝트 기간: 2026.03.18 ~ 2026.03.31
- 프로젝트 주제: "어떤 고객에게 어떤 프로모션을 제공해야 실제 구매로 이어지는가?"를 분석하고, 이를 Tableau로 시각화하는 프로젝트

---

## 프로젝트 배경
스타벅스는 최근 **거래 건수 감소로 인해 매출이 약 2% 하락**했으며,
**평균 객단가 상승만으로는 성장을 방어하기 어려운 상황**이다.

따라서 단순 매출 증대보다, 
**고객의 실제 거래 전환율(transaction)을 높이는 것**이 핵심 과제가 되었다.\
즉, 어떤 고객에게 어떤 프로모션을 어떤 방식으로 제공해야 
실제 구매로 이어지는지를 분석하는 것이 중요하다.

본 프로젝트에서는 스타벅스 데이터를 활용해 고객 정보, 오퍼 정보, 이벤트 로그를 종합적으로 분석하고, \
프로모션 반응과 실제 구매 행동의 차이를 파악하고자 한다.

---

## 대시보드 사용자(페르소나 설정)
**스타벅스 프로모션 담당 마케팅 분석가**

### 주요 관심사
1. 오퍼를 통해 실제 거래(transaction)를 얼마나 증가시킬 수 있는가?
2. 오퍼 → 구매 전환 구조를 직관적으로 파악할 수 있는가?
3. 고객 세그먼트별 반응 차이를 기반으로 타겟 전략을 수립할 수 있는가?
> 튜터님의 7번 이슈 예시. 추후 자세한 회의를 통해 설정 

---

## 프로젝트 목표
본 프로젝트의 핵심 목표는\
**고객 특성과 프로모션 특성에 따라 구매 전환율을 높이는 요인을 파악하는 것**이다.

구체적으로는 **어떤 고객(고객 세그먼트별)이 어떤 프로모션에 반응하고, 
그 반응이 실제 구매로 얼마나 이어지는지**를 분석한다.\
이때 고객 세그먼트는 단순한 인구통계 정보(성별, 연령, 소득)뿐만 아닌 
행동 기반 특성까지 포함하여 확장적으로 정의한다.

예를 들어,\
신규 고객 / 기존 고객\
일반 고객 / 가치 높은 고객(VIP)\
핵심 고객 / 잠재 고객 / 이탈 위험 고객\
등의 세그먼트를 후보로 두고, 구체적인 세그먼트 기준은 분석 과정에서 데이터 특성에 맞게 확정할 예정이다.

---

## 최종 산출물
스타벅스 이용 데이터를 기반으로 고객 특성, 오퍼 특성, 행동 데이터를 통합 분석해\
**구매 전환율에 영향을 주는 요인을 시각화**하고,\
**마케팅 전략 수립을 지원하는 Tableau 대시보드**를 제작한다.


### 프로젝트 확장성
가능하다면 **ROI(투자 대비 수익)** 관점까지 확장해, 
어떤 프로모션이 단순 반응률을 넘어\
**실제로 수익성 있는 프로모션인지**까지 확인하는 것을 목표로 한다.

---

### 프로젝트 과정
1. **문제 정의**
   - 분석 목적과 핵심 비즈니스 문제를 설정
2. **KPI 설정**
   - 반응률, 전환률, 구매 관련 지표 정의
3. **데이터 이해 및 전처리**
   - 고객, 오퍼, 이벤트 로그 데이터 구조 파악 및 정리
4. **분석 수행**
   - 고객/오퍼/채널 관점의 EDA 및 인사이트 도출
5. **대시보드 제작**
   - KPI 중심의 Tableau 시각화 구성
6. **마케팅 전략 제안**
   - 분석 결과를 바탕으로 실행 가능한 방향 제시

  - 일반 고객
  - VIP 고객
3. 행동 기반 세그먼트

  - 잠재 고객
  - 이탈 위험 고객
